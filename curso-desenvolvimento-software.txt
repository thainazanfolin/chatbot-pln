    <h1>Estrutura do Curso</h1>
    <ul>
        <li>Modalidade: Presencial com 20% da carga horária oferecida remotamente (aulas remotas síncronas).</li>
        <li>Carga Horária Total: 2.400 horas (2880 aulas de 50 minutos cada).</li>
        <li>Estágio Supervisionado: 240 horas obrigatórias.</li>
        <li>Duração: 6 semestres (mínimo de 3 anos, máximo de 5 anos).</li>
        <li>Vagas: 40 por turno (noturno).</li>
        <li>Forma de Acesso: Vestibular (prova única com questões do Ensino Médio) ou processo seletivo para vagas remanescentes.</li>
    </ul>

    <h1>Semestres e Disciplinas</h1>

    <h2>Primeiro Semestre - 1º semestre</h2>
    <ul>
        <li>Algoritmo e Lógica de Programação: Desenvolver competências para distinguir e aplicar metodologias de desenvolvimento de software, além de criar algoritmos e programas de computador, aplicando conceitos de lógica de programação.</li>
        <li>Desenvolvimento Web I: Introduzir os alunos ao desenvolvimento de aplicações web, utilizando linguagens de marcação e programação para criar páginas web estáticas e dinâmicas.</li>
        <li>Design Digital: Ensinar os princípios de design digital, focando na criação de interfaces gráficas, experiência do usuário e interação com sistemas computacionais.</li>
        <li><strong>Engenharia de Software I: Apresentar os conceitos básicos de engenharia de software, incluindo o ciclo de vida do software e as metodologias de desenvolvimento.</li>
        <li><strong>Modelagem de Banco de Dados: Introduzir o aluno à modelagem de dados, focando em banco de dados relacionais e as técnicas para projetar estruturas de armazenamento eficientes.</li>
        <li>Sistemas Operacionais e Redes de Computadores: Ensinar os fundamentos de sistemas operacionais e redes, preparando os alunos para compreender as interações entre hardware e software.</li>
    </ul>

    <h2>Segundo Semestre - 2º semestre</h2>
    <ul>
        <li><strong>Técnica de Programação I:</strong> Desenvolver habilidades em programação orientada a objetos, com foco em abstração, encapsulamento, herança e polimorfismo, além de interfaces gráficas e controle de exceções.</li>
        <li><strong>Desenvolvimento Web II:</strong> Avançar no desenvolvimento web, trabalhando com frameworks, banco de dados e criando APIs para aplicações.</li>
        <li><strong>Matemática para Computação:</strong> Prover os fundamentos matemáticos para apoiar o desenvolvimento de algoritmos e soluções computacionais.</li>
        <li><strong>Engenharia de Software II:</strong> Aprofundar os conceitos de engenharia de software, com ênfase no gerenciamento de requisitos, design e validação de software.</li>
        <li><strong>Banco de Dados Relacional:</strong> Ensinar como trabalhar com bancos de dados relacionais, focando em consultas SQL e manipulação de dados.</li>
        <li><strong>Estrutura de Dados:</strong> Introduzir as principais estruturas de dados (listas, pilhas, filas, árvores) e suas implementações.</li>
    </ul>

    <h2>Terceiro Semestre - 3º semestre</h2>
    <ul>
        <li><strong>Técnica de Programação II:</strong> Aplicar padrões de projeto no desenvolvimento de aplicações orientadas a objetos.</li>
        <li><strong>Desenvolvimento Web III:</strong> Focar no desenvolvimento de aplicações web escaláveis e seguras, utilizando práticas de integração e entrega contínua.</li>
        <li><strong>Álgebra Linear:</strong> Fornecer uma base sólida em álgebra linear para auxiliar na resolução de problemas computacionais e em machine learning.</li>
        <li><strong>Gestão Ágil de Projetos de Software:</strong> Ensinar metodologias ágeis de gestão de projetos, como Scrum e Kanban, para o desenvolvimento de software.</li>
        <li><strong>Banco de Dados Não-Relacional:</strong> Trabalhar com bancos de dados NoSQL, entendendo suas características e aplicações.</li>
        <li><strong>Interação Humano-Computador:</strong> Ensinar princípios de design de interfaces e interação com sistemas, focando na experiência do usuário.</li>
    </ul>

    <h2>Quarto Semestre - 4º semestre</h2>
    <ul>
        <li><strong>Integração e Entrega Contínua:</strong> Ensinar as melhores práticas para integração e entrega contínua de software, incluindo versionamento e automação de testes.</li>
        <li><strong>Laboratório de Desenvolvimento Web:</strong> Oferecer um ambiente prático para aplicar conceitos de desenvolvimento web, com foco em projetos práticos.</li>
        <li><strong>Internet das Coisas (IoT):</strong> Introduzir o conceito de IoT e capacitar os alunos para desenvolver sistemas conectados e inteligentes.</li>
        <li><strong>Programação para Dispositivos Móveis I:</strong> Ensinar o desenvolvimento de aplicações móveis, com foco em Android e iOS.</li>
        <li><strong>Estatística Aplicada:</strong> Prover conhecimentos de estatística para análise de dados, aplicáveis a projetos de software e machine learning.</li>
        <li><strong>Experiência do Usuário (UX):</strong> Aprofundar os conceitos de UX, focando em como melhorar a interação e a satisfação dos usuários.</li>
    </ul>

    <h2>Quinto Semestre - 5º semestre</h2>
    <ul>
        <li><strong>Computação em Nuvem I:</strong> Ensinar os fundamentos de computação em nuvem e infraestrutura de servidores virtuais.</li>
        <li><strong>Aprendizagem de Máquina:</strong> Introduzir o aluno aos conceitos e algoritmos de machine learning, aplicando-os a problemas reais.</li>
        <li><strong>Laboratório de Desenvolvimento Móvel:</strong> Aplicar as técnicas de desenvolvimento de aplicativos móveis em um ambiente de laboratório prático.</li>
        <li><strong>Programação para Dispositivos Móveis II:</strong> Avançar no desenvolvimento de aplicações móveis, integrando novas tecnologias e frameworks.</li>
        <li><strong>Segurança no Desenvolvimento de Aplicações:</strong> Ensinar boas práticas de segurança no desenvolvimento de software, protegendo dados e garantindo a integridade de sistemas.</li>
    </ul>

    <h2>Sexto Semestre - 6º semestre</h2>
    <ul>
        <li><strong>Computação em Nuvem II:</strong> Aprofundar os conceitos de computação em nuvem, com foco em arquiteturas escaláveis e automação.</li>
        <li><strong>Processamento de Linguagem Natural (PLN):</strong> Introduzir os alunos às técnicas e ferramentas de PLN para desenvolver sistemas que entendem e processam linguagem humana.</li>
        <li><strong>Mineração de Dados:</strong> Ensinar técnicas de mineração de dados para extrair informações valiosas de grandes conjuntos de dados.</li>
        <li><strong>Qualidade e Testes de Software:</strong> Focar na garantia da qualidade de software por meio de testes automatizados e metodologias de validação.</li>
    </ul>

    <h1>Certificações e Microcertificações</h1>
    <ul>
        <li>O aluno obtém 6 Microcertificações e 3 Certificações Intermediárias ao longo do curso, garantindo feedback contínuo de desempenho.</li>
        <li>Ao final, o aluno recebe o diploma de Tecnólogo em Desenvolvimento de Software Multiplataforma.</li>
        <li><strong>Exemplo de Microcertificações:</strong> Front-End Básico, Design Patterns Básico, UX Básico.</li>
        <li><strong>Certificações Intermediárias:</strong> Desenvolvedor Front-End, Desenvolvedor Back-End, Desenvolvedor para Dispositivos Móveis.</li>
    </ul>

    <h1>Regras da Faculdade</h1>
    <ul>
        <li><strong>Estágio Curricular Supervisionado:</strong> 240 horas obrigatórias.</li>
        <li><strong>Projeto Integrador:</strong> O portfólio digital será construído ao longo dos semestres e substitui o Trabalho de Graduação.</li>
        <li>Os projetos desenvolvidos nos últimos semestres são utilizados para compor esse portfólio.</li>
        <li><strong>Horas Complementares:</strong> Não especificado diretamente, mas há uma forte ênfase em projetos interdisciplinares e em pesquisa, que podem ser parte das atividades complementares
</body>
